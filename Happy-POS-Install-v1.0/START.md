# 🧾 Happy POS — คู่มือการติดตั้งและใช้งาน

## วิธีติดตั้ง (ครั้งแรกเท่านั้น)

```bash
# 1. ติดตั้ง dependencies
npm install

# 2. สร้างฐานข้อมูลเริ่มต้น (มีสินค้าตัวอย่าง)
npm run init-db
```

## เริ่มใช้งาน

```bash
npm run dev
```

จากนั้นเปิด browser ไปที่ **http://localhost:3000**

---

## ดูจากมือถือ (ผ่าน Wi-Fi เดียวกัน)

1. หา IP ของเครื่องที่รัน server:
   - **Windows:** เปิด CMD → พิมพ์ `ipconfig` → ดูที่ "IPv4 Address"
   - **Mac/Linux:** เปิด Terminal → พิมพ์ `ifconfig` หรือ `ip addr`

2. เปิด browser บนมือถือ → พิมพ์ `http://192.168.x.x:3000`

---

## การเชื่อมต่อเครื่องพิมพ์ Thermal Wireless

1. เชื่อมต่อเครื่องพิมพ์ (เช่น Epson TM-T82, SNBC, Bixolon) กับ Wi-Fi เดียวกัน
2. ติดตั้ง driver ของเครื่องพิมพ์บนเครื่อง
3. ตั้งเป็น **Default Printer** ในระบบ
4. กดปุ่ม "พิมพ์ใบเสร็จ" → เลือก printer และ Paper Size = 80mm

## ฟีเจอร์ทั้งหมด

| หน้า | ฟีเจอร์ |
|------|---------|
| 🛒 **ขาย (POS)** | เลือกสินค้า, ตะกร้า, ส่วนลด, เลือกสมาชิก, ชำระเงิน, พิมพ์ใบเสร็จ |
| 📊 **Dashboard** | ยอดขายวันนี้/เดือน, กราฟ, สินค้าขายดี, สต็อกใกล้หมด |
| 📦 **สินค้า/เมนู** | เพิ่ม/แก้ไข/ลบ, จัดหมวดหมู่, บาร์โค้ด |
| 🗄️ **คลังสินค้า** | รับสินค้าเข้า, ตัดออก, ปรับสต็อก, ประวัติการเคลื่อนไหว |
| 📈 **รายงาน** | ยอดขาย/กำไร, สินค้าขายดี, สัดส่วนการชำระ, Export CSV |
| 👥 **สมาชิก** | เพิ่มสมาชิก, แต้มสะสม, ประวัติการซื้อ, ระดับ (Bronze/Silver/Gold) |
| 🏷️ **โปรโมชั่น** | ส่วนลด %, ลดราคาตรง, กำหนดวันหมดอายุ, โค้ดส่วนลด |
| ⚙️ **ตั้งค่า** | ข้อมูลร้าน, ตั้งค่าใบเสร็จ, ระบบแต้มสะสม, แจ้งเตือนสต็อก |

## สำรองข้อมูล

ข้อมูลทั้งหมดอยู่ในโฟลเดอร์ `data/` — copy โฟลเดอร์นี้เพื่อสำรองข้อมูล

```
data/
  settings.json    ← ตั้งค่าระบบ
  products.json    ← สินค้า/เมนู
  categories.json  ← หมวดหมู่
  sales.json       ← ประวัติการขาย
  members.json     ← สมาชิก
  promotions.json  ← โปรโมชั่น
  stock_logs.json  ← ประวัติสต็อก
```

## Tech Stack

- **Next.js 14** (App Router) — Web framework
- **TypeScript** — Type safety
- **Tailwind CSS** — Styling
- **JSON Files** — Local database (offline-first, ไม่ต้อง internet)
- **Recharts** — กราฟและชาร์ต
- **Browser Print API** — พิมพ์ใบเสร็จ Thermal
