หลังจากติดตั้งเสร็จ เปิด Happy POS แล้วตั้งค่าพื้นฐานตามนี้:

1
ตั้งค่าข้อมูลร้าน

ไปที่เมนู ⚙️ ตั้งค่า → แท็บ "ข้อมูลร้าน"

กรอก: ชื่อร้าน, ที่อยู่, เบอร์โทร, เลขผู้เสียภาษี (ถ้ามี), PromptPay ID (เบอร์โทรหรือเลขบัตรประชาชน)

2
ตั้งค่าใบเสร็จ

แท็บ "ใบเสร็จ" → ใส่ข้อความหัวใบเสร็จ (เช่น "ยินดีต้อนรับ") และข้อความท้ายใบเสร็จ (เช่น "ขอบคุณที่ใช้บริการ")

3
เพิ่มหมวดหมู่สินค้า

ไปที่เมนู 📦 สินค้า → คลิก "จัดการหมวดหมู่" → เพิ่มหมวดหมู่ (เช่น เครื่องดื่ม, อาหาร, ขนม)

4
เพิ่มสินค้า/เมนู

หน้า 📦 สินค้า → คลิก "+ เพิ่มสินค้า" → กรอกชื่อ, ราคา, ต้นทุน, เลือกหมวดหมู่

5
ตั้งค่าเครื่องพิมพ์ (ถ้ามี)

แท็บ "เครื่องพิมพ์" → เพิ่มเครื่องพิมพ์ → เลือกชนิดการเชื่อมต่อ → ทดสอบพิมพ์

(ดูรายละเอียดที่หัวข้อ การติดตั้งเครื่องพิมพ์)

🛒 หน้าจอขายสินค้า (POS)

หน้านี้คือหน้าหลักที่ใช้ทุกวันในการขายสินค้า แบ่งเป็น 2 ส่วน:

ส่วนซ้าย — เลือกสินค้า

🔍 ช่องค้นหา: พิมพ์ชื่อสินค้า หรือสแกนบาร์โค้ด จะค้นหาให้อัตโนมัติ

📂 แถบหมวดหมู่: คลิกเลือกหมวดหมู่เพื่อกรองแสดงเฉพาะสินค้าในหมวดนั้น

📦 ตารางสินค้า: คลิกที่สินค้าเพื่อเพิ่มเข้าตะกร้า (คลิกซ้ำ = เพิ่มจำนวน)

💡
เคล็ดลับ: ถ้าใช้เครื่องสแกนบาร์โค้ด แค่สแกนก็จะเพิ่มสินค้าเข้าตะกร้าให้เลย ไม่ต้องพิมพ์อะไร!

ส่วนขวา — ตะกร้าสินค้า

📋 รายการในตะกร้า: แสดงสินค้าที่เลือก + จำนวน + ราคารวม

🔢 ปุ่ม +/- : เพิ่มหรือลดจำนวนแต่ละรายการ

📝 โน้ตรายการ: คลิกไอคอน 📝 ข้างรายการเพื่อเพิ่มหมายเหตุ (เช่น "ไม่ใส่น้ำตาล", "เผ็ดมาก")

🗑️ ลบรายการ: คลิกปุ่มกากบาท ✕ เพื่อลบรายการ

🔢 หมายเลขโต๊ะ: กรอกหมายเลขโต๊ะ (สำหรับร้านอาหาร)

ขั้นตอนการขาย (ทำทุกวัน)

1
เลือกสินค้า — คลิกสินค้าจากตาราง หรือสแกนบาร์โค้ด
2
ปรับจำนวน — กด +/- เพื่อเพิ่มลดจำนวน, เพิ่มโน้ตถ้าต้องการ
3
เลือกสมาชิก (ถ้ามี) — คลิก "เลือกสมาชิก" เพื่อสะสมแต้ม
4
ใส่ส่วนลด (ถ้ามี) — คลิก "ส่วนลด" เพื่อเลือกโปรโมชั่นหรือใส่โค้ด
5
กดปุ่ม "ชำระเงิน" — เลือกวิธีชำระ → กดยืนยัน → พิมพ์ใบเสร็จ

💰 การชำระเงิน

เมื่อกดปุ่ม "ชำระเงิน" จะมีหน้าต่างให้เลือกวิธีชำระ:

วิธีชำระเงินที่รองรับ

วิธีคำอธิบายหมายเหตุ
💵 เงินสดลูกค้าจ่ายเงินสดใส่จำนวนเงินที่รับ → ระบบคำนวณเงินทอนให้ มีปุ่มลัด 20, 50, 100, 500, 1000
📱 QR PromptPayแสดง QR Code ให้ลูกค้าสแกนจ่ายผ่าน PromptPayต้องตั้งค่า PromptPay ID ในหน้าตั้งค่าก่อน ระบบจะสร้าง QR พร้อมยอดเงินอัตโนมัติ
💳 บัตรเครดิต/เดบิตลูกค้าจ่ายผ่านบัตรใช้เครื่องรูดบัตรแยก แล้วเลือกวิธีนี้เพื่อบันทึกยอด
🔄 อื่นๆวิธีชำระอื่น (เช่น เช็ค, โอนธนาคาร)บันทึกยอดโดยไม่ระบุวิธีเฉพาะ
💡
QR PromptPay: ตั้งค่าครั้งเดียว! ไปที่ ⚙️ ตั้งค่า → ข้อมูลร้าน → กรอก PromptPay ID (เบอร์โทรศัพท์ หรือ เลขบัตรประชาชน) จากนั้นทุกครั้งที่เลือกชำระด้วย QR PromptPay ระบบจะสร้าง QR Code พร้อมยอดเงินให้อัตโนมัติ

หลังชำระเงินสำเร็จ

✅ บันทึกรายการขาย

✅ เปิดหน้าพิมพ์ใบเสร็จ (กด "พิมพ์" ใน browser)

✅ สะสมแต้มให้สมาชิก (ถ้าเลือกสมาชิกไว้)

✅ ส่งออเดอร์ไปหน้าจอครัว (ถ้าตั้งค่าสเตชันไว้)

✅ พิมพ์ KOT (Kitchen Order Ticket) อัตโนมัติ

💳 แยกบิล (Split Bill)

สำหรับกรณีที่ลูกค้าหลายคนต้องการจ่ายแยกกัน

3 โหมดแยกบิล

1. แบ่งเท่า (Equal): หารยอดเท่ากันตามจำนวนคน เช่น ยอด 300 บาท 3 คน = คนละ 100 บาท

2. กำหนดยอดเอง (Custom): ใส่ยอดเงินแต่ละคนเอง เช่น คนที่ 1 จ่าย 150, คนที่ 2 จ่าย 100, คนที่ 3 จ่ายส่วนที่เหลือ

3. แยกตามรายการ (By Items): เลือกว่าแต่ละรายการเป็นของใคร ระบบจะคำนวณยอดแต่ละคนให้

วิธีใช้

1
กดปุ่ม "ชำระเงิน" แล้วกดปุ่ม "แยกบิล"
2
เลือกจำนวนคน (2-6 คน) แล้วเลือกโหมดแยกบิล
3
ชำระเงินทีละคน — แต่ละคนเลือกวิธีชำระของตัวเองได้

📋 ออเดอร์ที่ค้างอยู่ (Open Orders)

สำหรับร้านอาหารที่ลูกค้านั่งทานก่อน จ่ายทีหลัง หรือกรณีที่ต้องพักออเดอร์ไว้ก่อน

วิธีพักออเดอร์

1
เลือกสินค้าใส่ตะกร้าตามปกติ + ใส่หมายเลขโต๊ะ
2
กดปุ่ม "พักออเดอร์" (แทนที่จะกดชำระเงิน)
3
ออเดอร์จะถูกส่งไปครัว + เก็บไว้ในรายการ "ออเดอร์ค้าง"

วิธีเรียกออเดอร์กลับมาชำระ

1
ไปที่เมนู 📋 ออเดอร์
2
คลิกที่ออเดอร์ที่ต้องการ (จะเห็นเลขโต๊ะ, ยอดรวม, เวลา)
3
กด "ชำระเงิน" หรือ "แก้ไข" เพื่อเพิ่มสินค้าก่อนชำระ
⚠️
ออเดอร์ที่ค้างนานเกิน 30 นาที จะแสดงสัญลักษณ์เตือนสีแดง

📦 จัดการสินค้า/เมนู

จัดการรายการสินค้าทั้งหมดของร้าน

เพิ่มสินค้าใหม่

1
ไปที่เมนู 📦 สินค้า → กดปุ่ม "+ เพิ่มสินค้า"
2
กรอกข้อมูล:

• ชื่อสินค้า — เช่น "กาแฟเย็น", "ข้าวผัดกุ้ง"

• ราคาขาย — ราคาที่ขายให้ลูกค้า

• ต้นทุน — ราคาต้นทุน (ใช้คำนวณกำไรในรายงาน)

• หมวดหมู่ — เลือกหมวดหมู่ที่สร้างไว้

• บาร์โค้ด — ใส่รหัสบาร์โค้ด (ไม่บังคับ สำหรับร้านค้าปลีก)

• ติดตามสต็อก — เปิด/ปิดการนับสต็อก

• สเตชันครัว — เลือกว่าสินค้านี้เตรียมที่สเตชันไหน (เช่น ครัว, บาร์)

3
กดปุ่ม "บันทึก"

แก้ไข/ลบสินค้า

แก้ไข: คลิกที่สินค้า → แก้ไขข้อมูล → กด "บันทึก"

ลบ: คลิกที่สินค้า → กดปุ่ม "ลบ" → ยืนยัน

💡
เคล็ดลับ: ถ้าสินค้าหมดชั่วคราว ไม่ต้องลบ! แค่ปิดสวิตช์ "เปิดขาย" ก็จะไม่แสดงในหน้า POS

📂 จัดการหมวดหมู่

จัดกลุ่มสินค้าให้หาง่ายในหน้าขาย

1
ไปที่ 📦 สินค้า → กดปุ่ม "จัดการหมวดหมู่"
2
กด "+ เพิ่มหมวดหมู่" → กรอก:

• ชื่อหมวดหมู่ — เช่น "เครื่องดื่ม", "อาหารจานหลัก"

• ไอคอน — เลือก Emoji เช่น ☕ 🍜 🍰

• สี — เลือกสีที่ต้องการ (8 สีให้เลือก)

📱
หมวดหมู่จะแสดงเป็นแถบด้านบนในหน้า POS ช่วยให้กดเลือกสินค้าได้เร็วขึ้น

🗄️ จัดการสต็อก/คลังสินค้า

ติดตามจำนวนสินค้าคงเหลือ สำหรับสินค้าที่เปิดติดตามสต็อก

การเคลื่อนไหวสต็อก

ประเภทใช้เมื่อไหร่ตัวอย่าง
📥 รับเข้าสั่งซื้อสินค้ามาเพิ่มรับน้ำดื่ม 100 ขวด
📤 ตัดออกสินค้าเสียหาย / หมดอายุตัดขนมหมดอายุ 5 ชิ้น
🔧 ปรับสต็อกนับจริงแล้วไม่ตรงปรับจากระบบ 50 เป็น 45

วิธีรับสินค้าเข้า

1
ไปที่เมนู 🗄️ คลังสินค้า
2
เลือกสินค้า → กดปุ่ม "📥 รับเข้า"
3
กรอกจำนวน + หมายเหตุ (เช่น "สั่งซื้อรอบวันที่ 15") → กด "บันทึก"
🔔
การแจ้งเตือนสต็อกต่ำ: เมื่อสินค้าเหลือน้อยกว่าค่าที่ตั้งไว้ (ค่าเริ่มต้น 10 ชิ้น) จะแจ้งเตือนที่หน้า Dashboard ตั้งค่าได้ที่ ⚙️ ตั้งค่า → ระบบ

👥 ระบบสมาชิก/Loyalty

สะสมแต้มให้ลูกค้าประจำ กระตุ้นให้กลับมาซื้อซ้ำ

ระดับสมาชิก (คำนวณอัตโนมัติ)

ระดับเงื่อนไขสัญลักษณ์
🥉 Bronzeยอดซื้อสะสม 0 - 4,999 บาทสีทองแดง
🥈 Silverยอดซื้อสะสม 5,000 - 19,999 บาทสีเงิน
🥇 Goldยอดซื้อสะสม 20,000 บาทขึ้นไปสีทอง

วิธีเพิ่มสมาชิก

1
ไปที่เมนู 👥 สมาชิก → กด "+ เพิ่มสมาชิก"
2
กรอก: ชื่อ, เบอร์โทร (จำเป็น), อีเมล (ไม่บังคับ), หมายเหตุ
3
กด "บันทึก" — สมาชิกจะเริ่มต้นที่ระดับ Bronze

วิธีใช้สมาชิกตอนขาย

ที่หน้า POS → กดปุ่ม "เลือกสมาชิก" → ค้นหาชื่อหรือเบอร์โทร → เลือกสมาชิก

เมื่อชำระเงินเสร็จ ระบบจะสะสมแต้มให้อัตโนมัติ

การตั้งค่าแต้มสะสม

ไปที่ ⚙️ ตั้งค่า → แท็บ "Loyalty"

• แต้มต่อบาท: ทุกๆ กี่บาทได้ 1 แต้ม (ค่าเริ่มต้น: 25 บาท = 1 แต้ม)

• มูลค่าแต้ม: 1 แต้มใช้แลกส่วนลดกี่บาท (ค่าเริ่มต้น: 1 แต้ม = 1 บาท)

🏷️ โปรโมชั่น/ส่วนลด

สร้างโปรโมชั่นหลากหลายรูปแบบ

ประเภทโปรโมชั่น

ประเภทคำอธิบายตัวอย่าง
% ลดเปอร์เซ็นต์ลดราคาเป็น % จากยอดรวมลด 10% สูงสุด 100 บาท
฿ ลดจำนวนเงินลดราคาเป็นจำนวนเงินตายตัวลด 50 บาท เมื่อซื้อครบ 300 บาท
🎁 ซื้อ X ได้ Yซื้อสินค้าครบจำนวน แถมฟรีซื้อ 2 แถม 1

วิธีสร้างโปรโมชั่น

1
ไปที่เมนู 🏷️ โปรโมชั่น → กด "+ เพิ่มโปรโมชั่น"
2
กรอกข้อมูล:

• ชื่อโปรโมชั่น — เช่น "ลดร้อนแรง 10%"

• ประเภท — เลือกจาก 3 แบบข้างต้น

• ยอดขั้นต่ำ — ซื้อครบเท่าไหร่ถึงจะใช้ได้ (ไม่บังคับ)

• โค้ดส่วนลด — ถ้าต้องการให้ลูกค้ากรอกโค้ด (ไม่บังคับ)

• วันที่เริ่ม/สิ้นสุด — กำหนดช่วงเวลา

3
เปิดสวิตช์ "เปิดใช้งาน" → กด "บันทึก"

วิธีใช้โปรโมชั่นตอนขาย

ที่หน้า POS → กดปุ่ม "ส่วนลด" → เลือกจาก:

• โปรโมชั่นที่ใช้ได้: แสดงรายการโปรโมชั่นที่ยอดถึงขั้นต่ำ

• กรอกโค้ด: ใส่โค้ดส่วนลดแล้วกด "ใช้"

• กำหนดเอง: ใส่ส่วนลดจำนวนเงินหรือ % เอง

🍳 หน้าจอครัว (Kitchen Display System — KDS)

แสดงออเดอร์ที่ส่งมาจากหน้าขาย ใช้แทนกระดาษสั่งอาหาร

ตั้งค่าสเตชันครัว (ทำครั้งแรก)

1
ไปที่ ⚙️ ตั้งค่า → แท็บ "สเตชัน"
2
กด "+ เพิ่มสเตชัน" → ตั้งชื่อ เช่น:

• "ครัวร้อน" — สำหรับอาหารจานหลัก

• "ครัวเย็น" — สำหรับสลัดและของเย็น

• "บาร์เครื่องดื่ม" — สำหรับเครื่องดื่ม

3
ไปที่หน้า 📦 สินค้า → แก้ไขสินค้าแต่ละตัว → เลือก "สเตชัน" ที่ต้องการ

วิธีใช้หน้าจอครัว

1
เปิด browser อีกเครื่อง/แท็บ → ไปที่ 🍳 ครัว → เลือกสเตชัน
2
เมื่อมีออเดอร์เข้ามา จะมี เสียงแจ้งเตือน + การ์ดสีแดง
3

💡
แนะนำ: ใช้แท็บเล็ตหรือจอมอนิเตอร์ราคาถูกวางไว้ที่ครัว เปิด browser ไว้ตลอด หน้าจอจะอัปเดตอัตโนมัติทุก 3 วินาที

📱 QR เมนูลูกค้า

ให้ลูกค้าสแกน QR Code เพื่อดูเมนูและสั่งอาหารเองผ่านมือถือ

ตั้งค่า QR เมนู

1
ไปที่ ⚙️ ตั้งค่า → แท็บ "QR เมนู"
2
กรอกหมายเลขโต๊ะที่ต้องการสร้าง QR
3
พิมพ์ QR Code ออกมาวางที่แต่ละโต๊ะ

เมื่อลูกค้าสแกน QR

1. ลูกค้าเห็นเมนูทั้งหมดบนมือถือ แบ่งตามหมวดหมู่

2. ลูกค้าเลือกสินค้า กดเพิ่มในตะกร้า

3. ลูกค้ากด "สั่ง" — ออเดอร์จะเข้าระบบอัตโนมัติ

4. ออเดอร์ปรากฏในหน้า 📋 ออเดอร์ + ส่งไปครัวทันที

⚠️
มือถือลูกค้าต้องเชื่อมต่อ Wi-Fi เดียวกันกับเครื่อง POS จึงจะใช้งานได้ แนะนำเปิด Wi-Fi สำหรับลูกค้าโดยเฉพาะ

🖨️ การติดตั้งเครื่องพิมพ์ Thermal

Happy POS รองรับเครื่องพิมพ์ความร้อน (Thermal Printer) สำหรับพิมพ์ใบเสร็จและ KOT ส่งครัว

ชนิดเครื่องพิมพ์ที่รองรับ

ชนิดการเชื่อมต่อคำอธิบายเหมาะกับ
🖥️ Browser Printใช้เครื่องพิมพ์ที่ติดตั้งในเครื่อง (ผ่านระบบ Windows/Mac)ร้านที่มีเครื่องพิมพ์ USB ต่อกับคอมพิวเตอร์โดยตรง
🌐 Network/IPเชื่อมต่อผ่าน Wi-Fi หรือ LAN โดยใช้ IP Addressร้านที่ต้องการเครื่องพิมพ์ไร้สาย หรือมีหลายจุดพิมพ์
🔌 USBเชื่อมต่อผ่านสาย USB โดยตรงร้านขนาดเล็ก มีเครื่องพิมพ์ 1 ตัว
📶 Bluetoothเชื่อมต่อผ่าน Bluetoothเครื่องพิมพ์พกพา ใช้กับแท็บเล็ต

ขนาดกระดาษ

ขนาดคำอธิบายแนะนำ
80mmกว้าง 80 มิลลิเมตร (มาตรฐาน)✅ แนะนำ — แสดงข้อมูลได้ครบ อ่านง่าย
58mmกว้าง 58 มิลลิเมตร (เล็ก)สำหรับเครื่องพิมพ์พกพาขนาดเล็ก

วิธีเพิ่มเครื่องพิมพ์ในระบบ

1
ไปที่ ⚙️ ตั้งค่า → แท็บ "เครื่องพิมพ์"
2
กด "+ เพิ่มเครื่องพิมพ์" → กรอก:

• ชื่อ — เช่น "เครื่องพิมพ์แคชเชียร์", "เครื่องพิมพ์ครัว"

• ชนิดการเชื่อมต่อ — เลือกจากตารางด้านบน

• IP Address + Port — กรณี Network (เช่น 192.168.1.50 : 9100)

• ขนาดกระดาษ — เลือก 80mm หรือ 58mm

3
กด "ทดสอบพิมพ์" เพื่อตรวจสอบว่าเชื่อมต่อได้
4
กด "บันทึก"

การตั้งค่าแบบ Browser Print (ง่ายที่สุด)

วิธีนี้ใช้ระบบพิมพ์ของ Windows/Mac ไม่ต้องตั้งค่า IP

1
ต่อเครื่องพิมพ์เข้าคอมพิวเตอร์ ด้วยสาย USB
2
ติดตั้ง Driver ที่มากับเครื่องพิมพ์ (ใส่ CD หรือดาวน์โหลดจากเว็บผู้ผลิต)
3
ตั้งเป็น Default Printer

Windows: Settings → Devices → Printers & scanners → เลือกเครื่องพิมพ์ → Set as default

Mac: System Preferences → Printers & Scanners → เลือกเครื่องพิมพ์ → Set Default

4
ใน Happy POS → เพิ่มเครื่องพิมพ์ → เลือกชนิด "Browser Print" → บันทึก

การตั้งค่าแบบ Network/IP (Wi-Fi / LAN)

วิธีนี้เหมาะกับเครื่องพิมพ์ที่เชื่อมต่อ Wi-Fi หรือต่อสาย LAN

1
เชื่อมต่อเครื่องพิมพ์กับ Wi-Fi เดียวกัน กับเครื่อง POS (ดูวิธีจากคู่มือเครื่องพิมพ์)
2
หา IP Address ของเครื่องพิมพ์

วิธีที่ 1: พิมพ์ Self-Test Page (กดปุ่ม Feed ค้าง 3 วินาที ตอนเปิดเครื่อง)

วิธีที่ 2: ดูจากหน้าตั้งค่า Router (เข้า 192.168.1.1 ใน browser)

ตัวอย่าง IP: 192.168.1.50

3
ใน Happy POS → เพิ่มเครื่องพิมพ์ → เลือกชนิด "Network/IP" → ใส่ IP + Port 9100
4
กด "ทดสอบพิมพ์" → ถ้าเครื่องพิมพ์พิมพ์ออกมา แสดงว่าเชื่อมต่อสำเร็จ!

🔧 แก้ปัญหาเครื่องพิมพ์

ปัญหาที่พบบ่อย

ปัญหาสาเหตุวิธีแก้
กดพิมพ์แล้วไม่มีอะไรออก เครื่องพิมพ์ไม่ได้เปิด หรือไม่ได้ต่อ ตรวจสอบว่าเครื่องพิมพ์เปิดอยู่ (มีไฟ LED สว่าง) + สาย USB/LAN เสียบแน่น
พิมพ์ออกมาเป็นภาษาต่างดาว Encoding ไม่ตรง ตรวจสอบว่าเครื่องพิมพ์รองรับภาษาไทย + ลองรีสตาร์ทเครื่องพิมพ์
พิมพ์ช้ามาก การเชื่อมต่อ Wi-Fi ไม่เสถียร ลองเปลี่ยนเป็น LAN (สาย) แทน Wi-Fi หรือย้ายเครื่องพิมพ์ให้ใกล้ Router
กระดาษเปล่าออกมา ใส่กระดาษกลับด้าน กระดาษ Thermal มีหน้าที่พิมพ์ได้แค่ด้านเดียว — ลองพลิกกลับด้าน
ไฟกระพริบสีแดง กระดาษหมด หรือฝาเปิดอยู่ เปลี่ยนม้วนกระดาษใหม่ + ปิดฝาให้สนิท
Network printer หาไม่เจอ IP เปลี่ยน หรือคนละ Wi-Fi ตรวจสอบว่าเครื่อง POS กับเครื่องพิมพ์อยู่ Wi-Fi/LAN เดียวกัน + ตรวจ IP ใหม่

วิธีหา IP เครื่องพิมพ์

วิธีที่ 1 — พิมพ์ Self-Test:

ปิดเครื่องพิมพ์ → กดปุ่ม FEED ค้างไว้ → เปิดเครื่อง → ปล่อยปุ่ม FEED เมื่อเครื่องเริ่มพิมพ์

จะพิมพ์หน้า Test ที่มี IP Address แสดงอยู่

วิธีที่ 2 — ดูจาก Router:

เปิด browser → พิมพ์ 192.168.1.1 → ล็อกอิน → ดูรายการ "Connected Devices" → หาอุปกรณ์ที่ชื่อเป็นเครื่องพิมพ์

💡
แนะนำ: ตั้ง IP แบบ Static (คงที่) ให้เครื่องพิมพ์ เพื่อไม่ให้ IP เปลี่ยนเมื่อปิดเปิดเครื่อง — ตั้งค่าได้ที่ Router หรือที่ตัวเครื่องพิมพ์

ธีมสี

Happy POS มีธีมสี 10 แบบให้เลือก: ส้มคลาสสิก, น้ำเงิน, เขียว, ชมพู, ม่วง, น้ำตาล, มืด, เทา, แดง, ทอง

หรือเลือก "กำหนดเอง" แล้วเลือกสีที่ชอบจาก Color Picker

📱 ใช้งานจากมือถือ/แท็บเล็ต

เปิด Happy POS จากมือถือหรือแท็บเล็ตที่ต่อ Wi-Fi เดียวกัน

1
ต่อ Wi-Fi เดียวกัน กับเครื่องที่รัน Happy POS
2
หา IP ของเครื่อง POS:

• Windows: เปิด CMD → พิมพ์ ipconfig → ดูที่ "IPv4 Address"

• Mac: System Preferences → Network → ดู IP Address

3
เปิด browser บนมือถือ → พิมพ์ http://[IP]:21300 เช่น http://192.168.1.100:21300

แนะนำการใช้งานหลายจุด

📱 แท็บเล็ตที่ครัว: เปิดหน้า "ครัว" → เลือกสเตชัน → แสดงออเดอร์แบบ Realtime

📱 มือถือพนักงาน: เปิดหน้า "POS" → รับออเดอร์ที่โต๊ะลูกค้าได้เลย

💻 คอมพิวเตอร์แคชเชียร์: หน้าหลักสำหรับขาย + พิมพ์ใบเสร็จ

💾 สำรองข้อมูล

ข้อมูลทั้งหมดเก็บในเครื่องคอมพิวเตอร์ของคุณ ควรสำรองข้อมูลเป็นประจำ

ที่เก็บข้อมูล

Windows: C:\Users\[ชื่อผู้ใช้]\HappyPOS\data\

Mac: ~/HappyPOS/data/

วิธีสำรองข้อมูล

1
ปิด Happy POS ก่อน (กด Ctrl+C ที่หน้าต่าง Terminal/CMD)
2
คัดลอกโฟลเดอร์ data ทั้งหมดไปเก็บในที่ปลอดภัย (USB, Google Drive, OneDrive)

วิธีกู้คืนข้อมูล

คัดลอกโฟลเดอร์ data ที่สำรองไว้ กลับไปวางทับโฟลเดอร์ data เดิม แล้วเปิด Happy POS

⚠️
สำคัญมาก! แนะนำให้สำรองข้อมูลอย่างน้อยสัปดาห์ละ 1 ครั้ง เพื่อป้องกันข้อมูลสูญหาย

❓ คำถามที่พบบ่อย

▸ Happy POS ต้องมีอินเทอร์เน็ตตลอดไหม?

ไม่ต้อง! ต้องมีอินเทอร์เน็ตเฉพาะตอนติดตั้งครั้งแรกเท่านั้น หลังจากนั้นใช้งานได้แม้ไม่มีอินเทอร์เน็ต

▸ ใช้ได้กับร้านค้าประเภทไหนบ้าง?

ใช้ได้กับร้านค้าทุกประเภท เช่น ร้านกาแฟ, ร้านอาหาร, ร้านค้าปลีก, ร้านสปา, ร้านขนม, ร้านเสริมสวย — ฟีเจอร์ออกแบบมาให้ยืดหยุ่นกับธุรกิจ SME ทั่วไป

▸ ข้อมูลลูกค้าปลอดภัยไหม?

ข้อมูลทั้งหมดเก็บในเครื่องของคุณเท่านั้น ไม่ส่งไปที่ไหน ไม่มี Cloud ไม่มีบุคคลที่สามเข้าถึงได้

▸ เปิดหลายเครื่องพร้อมกันได้ไหม?

ได้! เครื่องหลัก (เครื่องที่รัน server) จะเป็นแม่ข่าย อุปกรณ์อื่นในเครือข่าย Wi-Fi เดียวกันสามารถเปิด browser เข้ามาใช้งานพร้อมกันได้ เช่น แท็บเล็ตที่ครัว, มือถือพนักงาน

▸ ต้องเปิดคอมพิวเตอร์ค้างไว้ตลอดไหม?

ใช่ เครื่องที่รัน Happy POS ต้องเปิดอยู่ตลอดเวลาที่ใช้งาน ถ้าปิดเครื่องหรือปิดหน้าต่าง Terminal/CMD ระบบจะหยุดทำงาน เปิดใหม่ได้โดยดับเบิลคลิก shortcut "Happy POS" บน Desktop

▸ สามารถเปลี่ยนราคาสินค้าระหว่างวันได้ไหม?

ได้เลย! ไปที่หน้า "สินค้า" → คลิกที่สินค้า → แก้ราคา → บันทึก ราคาจะเปลี่ยนทันที ไม่ต้องรีสตาร์ทระบบ

▸ ยกเลิกรายการขายที่ผิดพลาดได้ไหม?

รายการที่ยังไม่ได้ชำระสามารถลบออกจากตะกร้าได้ทันที แต่รายการที่ชำระแล้วจะบันทึกเป็นประวัติไม่สามารถลบได้ เพื่อความถูกต้องของรายงาน

▸ อัปเดต Happy POS เวอร์ชันใหม่ทำอย่างไร?

รัน install script อีกครั้ง (install-windows.bat หรือ install-mac.command) ระบบจะอัปเดตโค้ดให้อัตโนมัติ ข้อมูลร้านค้าและรายการขายจะไม่ถูกลบ
