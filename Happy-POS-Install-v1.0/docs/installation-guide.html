🪟 Windows: install-windows.bat 🍎 Mac: install-mac.command

1 วิธีที่ 1: ติดตั้งจากไฟล์ที่ได้รับ — USB / ZIP (แนะนำ)

วิธีนี้ง่ายที่สุดและเร็วที่สุด! ไม่ต้องดาวน์โหลดจากอินเทอร์เน็ต (ยกเว้น Node.js ถ้ายังไม่มี)

💡
วิธีนี้เหมาะสำหรับ: ได้รับไฟล์ Happy POS ทั้งโฟลเดอร์ผ่าน USB, ZIP, หรือคัดลอกจากเครื่องอื่น
ไม่ต้องติดตั้ง Git เพราะใช้ไฟล์จากเครื่องโดยตรง
1
เตรียมไฟล์ Happy POS

ถ้าได้รับเป็นไฟล์ ZIP → คลิกขวาแล้วเลือก "Extract All..." (แตกไฟล์ทั้งหมด)

ถ้าได้รับจาก USB → คัดลอกโฟลเดอร์ happy-pos ทั้งหมดมาไว้ในเครื่อง

2
เปิดโฟลเดอร์ Happy POS แล้วหาไฟล์ install-windows.bat

  ├── 📁 app/
  ├── 📁 components/
  └── 📁 lib/
✨
Script จะตรวจพบ package.json ในโฟลเดอร์เดียวกันอัตโนมัติ และจะใช้ไฟล์จากเครื่องเลย — ไม่ต้องดาวน์โหลดจาก GitHub, ไม่ต้องติดตั้ง Git

ดับเบิลคลิกที่ไฟล์ install-windows.bat

Script จะขอสิทธิ์ผู้ดูแลระบบอัตโนมัติ — ถ้ามีหน้าต่างถาม ให้กด "Yes" หรือ "ใช่"

⚠️
ถ้า Windows แจ้งเตือน "Windows protected your PC"
คลิก "More info" แล้วคลิก "Run anyway"

รอ — ไม่ต้องทำอะไรเลย! (ประมาณ 3-5 นาที)

ระบบจะทำทุกอย่างให้อัตโนมัติ:

✅ ตรวจสอบและติดตั้ง Node.js (ถ้ายังไม่มี)

⏭️ ข้ามการติดตั้ง Git (ไม่จำเป็นเมื่อมีไฟล์ในเครื่อง!)

✅ คัดลอกไฟล์ไปที่ C:\Users\[ชื่อ]\HappyPOS

✅ ติดตั้ง packages ที่จำเป็น

✅ สร้างฐานข้อมูลเริ่มต้น

✅ สร้าง shortcut "Happy POS" บน Desktop

✅ เปิด Happy POS ให้อัตโนมัติทันที!

🎉
เมื่อติดตั้งเสร็จ เบราว์เซอร์จะเปิด Happy POS ให้อัตโนมัติ — ไม่ต้องกดอะไรเพิ่มเลย! ครั้งต่อไปแค่ดับเบิลคลิก "Happy POS" บน Desktop

1
ดาวน์โหลดเฉพาะไฟล์ install-windows.bat

ดาวน์โหลดจาก GitHub หรือขอจากผู้ดูแลระบบ

2
วางไฟล์ไว้ที่ไหนก็ได้ แล้วดับเบิลคลิก

Script จะขอสิทธิ์ผู้ดูแลระบบอัตโนมัติ และตรวจพบว่าไม่มี package.json ข้างๆ จะดาวน์โหลดจาก GitHub อัตโนมัติ

⚠️
ถ้า Windows แจ้งเตือน "Windows protected your PC"
คลิก "More info" แล้วคลิก "Run anyway"

รอให้ติดตั้งเสร็จ (ประมาณ 5-10 นาที)

ระบบจะ:

✅ ติดตั้ง Node.js + Git อัตโนมัติ (ถ้ายังไม่มี)

✅ ดาวน์โหลด Happy POS จาก GitHub

✅ ติดตั้ง packages + สร้างฐานข้อมูล + สร้าง shortcut

🎉
เมื่อเห็นข้อความ "ติดตั้ง Happy POS สำเร็จแล้ว!" แสดงว่าเสร็จเรียบร้อย!

3 วิธีที่ 3: ติดตั้งด้วยตัวเอง (step by step)

สำหรับผู้ที่ต้องการติดตั้งเอง หรือ script อัตโนมัติมีปัญหา

ขั้นตอนที่ 1: ติดตั้ง Node.js

1
เปิด browser ไปที่ https://nodejs.org

เลือกดาวน์โหลดปุ่มสีเขียว "LTS" (Long Term Support)

2
ดับเบิลคลิกไฟล์ที่ดาวน์โหลด

กด Next → Next → Next → Install → Finish (ใช้ค่าเริ่มต้นทั้งหมด)

3
ตรวจสอบ: เปิด Command Prompt แล้วพิมพ์
node -v

ถ้าเห็นเลขเวอร์ชัน เช่น v20.18.0 แสดงว่าสำเร็จ!

💡
วิธีเปิด Command Prompt: กดปุ่ม Win + R แล้วพิมพ์ cmd แล้วกด Enter

ขั้นตอนที่ 2: ติดตั้ง Git

1
เปิด browser ไปที่ https://git-scm.com

คลิก "Download for Windows"

2
ดับเบิลคลิกไฟล์ที่ดาวน์โหลด

กด Next ไปเรื่อยๆ จนกด Install → Finish (ใช้ค่าเริ่มต้นทั้งหมด)

ขั้นตอนที่ 3: ดาวน์โหลดและติดตั้ง Happy POS

1
เปิด Command Prompt แล้วพิมพ์คำสั่งนี้ทีละบรรทัด:
# ดาวน์โหลด Happy POS
git clone https://github.com/happydaddy/happy-pos.git %USERPROFILE%\HappyPOS

# เข้าไปในโฟลเดอร์
cd %USERPROFILE%\HappyPOS

# ติดตั้ง packages
npm install

# สร้างฐานข้อมูลเริ่มต้น
npm run init-db

▶ วิธีเปิดใช้งาน Happy POS

วิธีที่ 1: ดับเบิลคลิก shortcut (ง่ายที่สุด)

ดับเบิลคลิกที่ไอคอน "Happy POS" บน Desktop

ระบบจะเปิด server และเปิด browser ให้อัตโนมัติ

วิธีที่ 2: ดับเบิลคลิกไฟล์ start

ไปที่โฟลเดอร์ Happy POS แล้วดับเบิลคลิก start-happy-pos.bat

⚠️
อย่าปิดหน้าต่างสีดำ (Command Prompt)!
หน้าต่างนี้คือ server ของ Happy POS — ถ้าปิดจะใช้งานไม่ได้
เมื่อใช้งานเสร็จ กด Ctrl + C เพื่อปิด server
🌐
หลังจากเปิด server แล้ว สามารถเปิด browser เองได้ที่:
http://localhost:21300

📱 ดูจากมือถือ (ผ่าน Wi-Fi เดียวกัน)

1
ต่อมือถือกับ Wi-Fi เดียวกับคอมพิวเตอร์
2
หา IP ของคอมพิวเตอร์

เปิด Command Prompt แล้วพิมพ์:

ipconfig

ดูที่บรรทัด "IPv4 Address" เช่น 192.168.1.100

3
เปิด browser บนมือถือ

พิมพ์: http://192.168.1.100:21300

(เปลี่ยน IP ตามที่ได้จากขั้นตอนที่ 2)

❓ คำถามที่พบบ่อย (Windows)

ติดตั้งแล้วเปิดไม่ขึ้น ทำอย่างไร? +

1. ตรวจสอบว่า Node.js ติดตั้งสำเร็จ: เปิด CMD พิมพ์ node -v

2. ลองรัน npm install อีกครั้งในโฟลเดอร์ HappyPOS

3. ตรวจสอบว่า Firewall ไม่ได้บล็อก port 21300

"Windows protected your PC" ขึ้นมา ต้องทำอย่างไร? +

นี่เป็นเรื่องปกติสำหรับ script ที่ดาวน์โหลดจากอินเทอร์เน็ต

1. คลิก "More info" (ข้อมูลเพิ่มเติม)

2. คลิก "Run anyway" (รันต่อไป)

จะอัปเดต Happy POS ทำอย่างไร? +

รัน install-windows.bat อีกครั้ง ระบบจะตรวจสอบและอัปเดตให้อัตโนมัติ

ไม่มีอินเทอร์เน็ต ใช้งานได้ไหม? +

ได้! หลังจากติดตั้งเสร็จแล้ว Happy POS ทำงานแบบ offline ได้ทั้งหมด

ต้องมีอินเทอร์เน็ตเฉพาะตอนติดตั้งครั้งแรกเท่านั้น

ข้อมูลร้านค้าเก็บไว้ที่ไหน? +

ข้อมูลทั้งหมดเก็บในเครื่องคอมพิวเตอร์ของคุณที่:

C:\Users\[ชื่อผู้ใช้]\HappyPOS\data\

แนะนำให้สำรองข้อมูลโฟลเดอร์ data เป็นประจำ

เมื่อใช้งานเสร็จ กด Ctrl + C เพื่อปิด server
🌐
หลังจากเปิด server แล้ว สามารถเปิด browser เองได้ที่:
http://localhost:21300

📱 ดูจากมือถือ (ผ่าน Wi-Fi เดียวกัน)

1
ต่อมือถือกับ Wi-Fi เดียวกับ Mac
2
หา IP ของ Mac

คลิกที่ไอคอน Wi-Fi บน menu bar → กด Option ค้างไว้แล้วคลิก Wi-Fi

ดูที่ "IP Address" เช่น 192.168.1.100

หรือเปิด Terminal แล้วพิมพ์:

ipconfig getifaddr en0
3
เปิด browser บนมือถือ

พิมพ์: http://192.168.1.100:21300

(เปลี่ยน IP ตามที่ได้จากขั้นตอนที่ 2)

❓ คำถามที่พบบ่อย (Mac)

ดับเบิลคลิกไฟล์ .command แล้วไม่เปิด ทำอย่างไร? +

1. คลิกขวาที่ไฟล์ → เลือก "Open" → คลิก "Open" อีกครั้ง

2. ถ้ายังไม่ได้ ไปที่ System Preferences → Security & Privacy → คลิก "Open Anyway"

3. อีกวิธี: เปิด Terminal แล้วพิมพ์ chmod +x install-mac.command แล้วดับเบิลคลิกอีกครั้ง

Homebrew ติดตั้งนานมาก ปกติไหม? +

ปกติครับ! การติดตั้ง Homebrew ครั้งแรกอาจใช้เวลา 5-10 นาที ขึ้นอยู่กับอินเทอร์เน็ต

ถ้ามันดาวน์โหลด Xcode Command Line Tools ด้วยอาจใช้เวลานานขึ้น

ถามรหัสผ่าน แต่พิมพ์แล้วไม่เห็นอะไร ปกติไหม? +

ปกติครับ! ใน Terminal เมื่อพิมพ์รหัสผ่านจะไม่แสดงอะไรเลย (ไม่มี * หรือ ● ขึ้น)

พิมพ์รหัสผ่านให้ถูกต้องแล้วกด Enter ได้เลย

ไม่มีอินเทอร์เน็ต ใช้งานได้ไหม? +

ได้! หลังจากติดตั้งเสร็จแล้ว Happy POS ทำงานแบบ offline ได้ทั้งหมด

ต้องมีอินเทอร์เน็ตเฉพาะตอนติดตั้งครั้งแรกเท่านั้น

ข้อมูลร้านค้าเก็บไว้ที่ไหน? +

ข้อมูลทั้งหมดเก็บในเครื่อง Mac ของคุณที่:

~/HappyPOS/data/

แนะนำให้สำรองข้อมูลโฟลเดอร์ data เป็นประจำ
